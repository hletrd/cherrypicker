## 카드 특징 요약

### taptap 시리즈
삼성카드의 대표 라인업으로, 패키지 선택형 혜택 구조가 특징입니다. 매월 소비 패턴에 따라 패키지를 변경할 수 있어 유연한 혜택 운용이 가능합니다.

### iD 시리즈
맞춤형 할인 카드 라인업입니다. iD ON은 AI 기반 맞춤 할인, iD ALL은 생활밀착형 할인, iD SIMPLE은 무실적 기본 할인을 제공합니다. iD MOVE는 대중교통·통신 특화, iD PLUG-IN은 전기차·디지털구독 특화, iD VITA는 의료·건강 특화, iD STATION은 주유 특화 카드입니다.

### 마일리지 카드
& MILEAGE PLATINUM은 대한항공 스카이패스 마일리지 적립에 특화된 카드로, 전월실적 없이 마일리지가 적립됩니다.

### 제휴 카드
다이소 삼성카드는 다이소·배달앱·커피전문점에 특화된 생활 밀착형 제휴카드입니다. 무실적·무한도 1% 기본 할인이 강점입니다.
